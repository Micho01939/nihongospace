🎌 Lección 2: ¿Qué es el Hiragana? (ひらがな)

El Hiragana es la piedra angular del idioma japonés. Es el primer sistema de escritura que todo estudiante debe aprender y el alfabeto básico con el que desbloquearás el resto del idioma. Si el japonés fuera una casa, el Hiragana sería los cimientos.

🛠️ ¿Para qué se usa en el Japonés Real?

En una página web o un texto japonés real, verás el Hiragana mezclado con otros sistemas. Se utiliza específicamente para:

  • Palabras nativas: Vocabulario propio de Japón que no usa caracteres chinos (ej. Sakura / Cerezo 🌸 -> さくら).
  • Gramática y conectores: Unir las palabras en una frase (partículas) y conjugar los verbos.
  • Furigana (Guías de lectura): Pequeños caracteres de Hiragana que se colocan encima de los Kanjis complejos para enseñarle al usuario cómo se pronuncian.

Conociendo la vocal "A" (あ)

La あ (A) es el primer carácter del Hiragana. Es el origen de todo el silabario y la base para el resto de los sonidos.

🔊 Pronunciación: Sonido: Se pronuncia exactamente igual que la "A" en español (como en "casa" o "árbol").
